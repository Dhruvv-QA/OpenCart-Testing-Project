# OpenCart Testing Project

#  Overview
This project demonstrates end-to-end manual testing of an OpenCart e-commerce application. It includes test scenario design, detailed test case creation, execution, and real-time bug reporting using JIRA.

The project simulates a real QA workflow following Agile methodology.

## Modules Tested
- Login
- Registration
- Cart
- Checkout

## Work Done
- Designed test scenarios and test cases
- Executed test cases (positive, negative, edge cases)
- Identified and reported bugs using Jira
- Followed Agile methodology

## Tools Used
- Jira
- MS Excel

## Bug Reporting
- Identified defects during execution
- Reported bugs with proper steps, expected vs actual results
- Attached screenshots for clear evidence
- Maintained structured bug reports inside Bug_Reports folder
